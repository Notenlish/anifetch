# Commands to use when testing

## Cache
`anifetch --cache-list`

`anifetch --clear`

`anifetch --delete [cache_id]`

`anifetch example.mp4 --force-render`

## Animation
`anifetch example.mp4`

`anifetch example.mp4 --sound [SOUND_FILE]`

`anifetch [file_with_audio] --sound`

`anifetch example.mp4 -W 10`

`anifetch example.mp4 -H 20`

`anifetch example.mp4 -i 1`


## Centering
`anifetch example.mp4 --center`

## Neofetch
`anifetch example.mp4 --neofetch`

## Chafa Args
`anifetch example.mp4 -ca "--symbols wide --fg-only"`  --> should print with japanese characters

## quality
`anifetch example.mp4 --quality 2`

`anifetch example.mp4 --quality 10`


## Resizing

run `anifetch example.mp4` and resize the terminal
